# Nipmod Platform Connection Kits

This folder keeps every platform connection in the same shape.

Each kit answers four questions:

- What status can Nipmod honestly claim today?
- What command or proof checks that status?
- What does an agent or reviewer read first?
- What external approval is still needed before stronger wording is allowed?

Status labels:

| Status | Meaning |
| --- | --- |
| Live | Nipmod controls the path and the smoke or sync proof is passing. |
| MCP ready | The host can connect to the local Nipmod MCP server. |
| Under review | Nipmod prepared the work, but an outside owner must accept it. |
| Candidate | Good fit, but owner review has not happened yet. |
