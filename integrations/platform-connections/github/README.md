Use this wording:

> Nipmod is open on GitHub for public review, CI and developer access.

Do not claim:

> GitHub is the canonical package source for Nipmod.

https://github.com/nipmod/nipmod

```sh
git ls-remote https://github.com/nipmod/nipmod.git refs/heads/main
```
